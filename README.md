Softly é uma biblioteca CSS para simples aplicações PWA. Codificações de componentes pré-fabricados, focados na automatização de aplicações, priorizando desenvolvimentos juntamente com a ferramenta Apache Cordova. A biblioteca é composta com codificações arquitetadas através de classes CSS, acompanhe em nosso site a documentação do Softly.

https://softly.netlify.app/

Versão: 1.1.2

Extensões disponíveis: CSS e SCSS
